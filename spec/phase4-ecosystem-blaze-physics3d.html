Blaze Phase 4 – Ecosystem: Physics3D

Goal

3D rigid‑body physics engine with SoA layout, GJK/EPA collision detection, mesh colliders, BVH broad‑phase. Parallel, deterministic.

1. Components

@layout(soa)
pub struct RigidBody3D {
    pos: Vec3,
    vel: Vec3,
    orientation: Quat,
    angular_vel: Vec3,
    mass: f32,
    inv_mass: f32,
    inertia: Mat3,
    inv_inertia: Mat3,
    restitution: f32,
    friction: f32,
    body_type: BodyType,
    flags: BodyFlags,
}
pub struct Collider3D { shape: ColliderShape, local_pos: Vec3, local_orientation: Quat, category_bits: u16, mask_bits: u16 }
pub struct Contact3D { point: Vec3, normal: Vec3, penetration: f32, tangent1: Vec3, tangent2: Vec3, normal_impulse: f32, tangent_impulse1: f32, tangent_impulse2: f32, body_a: Entity, body_b: Entity }
