Blaze Phase 3b – Systems Library: Endianness

Previous: Binary | Next: File System

Goal

Host‑endianness detection and byte‑swapping functions for integers and floats.

1. Host Endianness

1.1 Endianness Enum

pub enum Endianness { Little, Big }

1.2 endianness Function

pub fn endianness() -> Endianness;

Compile‑time constant determined by target_endian. Zero‑cost.

2. Byte‑Swap Functions

All take value, return swapped value. Pure.

Integers

pub fn swap_u16(x: u16) -> u16;
pub fn swap_u32(x: u32) -> u32;
pub fn swap_u64(x: u64) -> u64;
pub fn swap_i16(x: i16) -> i16;
pub fn swap_i32(x: i32) -> i32;
pub fn swap_i64(x: i64) -> i64;

Floats

pub fn swap_f32(x: f32) -> f32;
pub fn swap_f64(x: f64) -> f64;

Reinterpret as unsigned integer, swap bytes, reinterpret back. Safe for all bit patterns (including NaN).

3. Testing

  • endianness() matches target.
  • Swap twice → original.
  • Known bit patterns: swap_u32(0x12345678) → 0x78563412.
  • Float swap: raw bits match integer swap.
  • Pure: callable from pure contexts.
