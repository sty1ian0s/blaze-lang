Goal

Provide compile‑time unit system using const‑generics. Defines Quantity and SI dimensions.

1. Dimensions

pub struct Dimensions {
    pub meters: i8,
    pub seconds: i8,
    pub kilograms: i8,
    pub amperes: i8,
    pub kelvins: i8,
    pub moles: i8,
    pub candelas: i8,
}

@copy, supports equality and compile‑time arithmetic.

Base Constants

pub const Metre: Dimensions = ...;
pub const Second: Dimensions = ...;
pub const Kilogram: Dimensions = ...;
pub const Ampere: Dimensions = ...;
pub const Kelvin: Dimensions = ...;
pub const Mole: Dimensions = ...;
pub const Candela: Dimensions = ...;

Derived Constants

Hertz, Newton, Pascal, Joule, Watt, Coulomb, Volt, Ohm, Farad, Tesla, Weber, Henry, Lumen.

2. Quantity<T, const D: Dimensions>

pub struct Quantity<T, const D: Dimensions>(T);

@copy if T is.

Arithmetic

  • Add/Sub: same dimension → same dimension.
  • Mul: dimensions add; Div: dimensions subtract.
  • Compile‑time dimension checks.

Conversion

pub fn convert<T, const S: Dimensions, const D: Dimensions>(q: Quantity<T, S>) -> Quantity<T, D>;

Best‑effort; static assert S == D for now. Actual conversion factors left for later.

3. Type Aliases

Convenient aliases: Metre<T>, Second<T>, Kilogram<T>, Hertz<T>, Newton<T>, etc.

4. Testing

  • Base quantities arithmetic; dimensions propagate.
  • Adding different dimensions fails at compile time.
  • Multiplication/division produce correct dimension tuples.
  • Conversion: same dimension works.
