Blaze Phase 4 – Ecosystem: Physics2D

Goal

2D rigid‑body physics engine built on ECS, SoA layout, and sliding‑window iteration. Pure, deterministic, auto‑parallelised.

1. Components

@layout(soa)
pub struct RigidBody2D {
    pos: Vec2,
    vel: Vec2,
    angle: f32,
    angular_vel: f32,
    mass: f32,
    inv_mass: f32,
    inertia: f32,
    inv_inertia: f32,
    restitution: f32,
    friction: f32,
    body_type: BodyType,
    flags: BodyFlags,
}
pub enum BodyType { Static, Dynamic, Kinematic }
pub struct Collider2D { shape: ColliderShape, local_pos: Vec2, local_angle: f32, category_bits: u16, mask_bits: u16 }
pub struct Contact { point: Vec2, normal: Vec2, penetration: f32, tangent: Vec2, normal_impulse: f32, tangent_impulse: f32, body_a: Entity, body_b: Entity }
