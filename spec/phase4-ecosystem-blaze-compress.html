Blaze Phase 4 – Ecosystem: Compress

Previous: CLI | Next: Config

Goal

Unified interface for compression/decompression algorithms: Deflate, Gzip, Zlib, Brotli, LZ4, Zstandard. Pure operations on byte slices; streaming adaptors.

1. Core Traits

pub trait Compressor {
    fn compress(&mut self, input: &[u8], output: &mut Vec<u8>) -> Result<(), CompressError>;
    fn finish(&mut self, output: &mut Vec<u8>) -> Result<(), CompressError>;
}
pub trait Decompressor {
    fn decompress(&mut self, input: &[u8], output: &mut Vec<u8>) -> Result<(), CompressError>;
    fn is_finished(&self) -> bool;
}

2. Supported Algorithms

  • DeflateEncoder/DeflateDecoder
  • GzipEncoder/GzipDecoder
  • ZlibEncoder/ZlibDecoder
  • BrotliEncoder (level 0‑11) / BrotliDecoder
  • Lz4Encoder (acceleration) / Lz4Decoder
  • ZstdEncoder (level 1‑22) / ZstdDecoder

3. Convenience Functions

One‑shot compress/decompress for each algorithm (e.g., compress_deflate, decompress_gzip).

4. Streaming

pub struct CompressWriter<W: Write> { writer: W, encoder: Box<dyn Compressor>, buf: Vec<u8> }
impl<W: Write> Write for CompressWriter<W> { … }
pub struct DecompressReader<R: Read> { reader: R, decoder: Box<dyn Decompressor>, buf: Vec<u8>, out_pos: usize }
impl<R: Read> Read for DecompressReader<R> { … }

5. Archive Support (feature `tar`)

pub fn archive_files(paths: &[&str], output: &str, compression: Compression) -> Result<(), CompressError>;
pub fn extract_archive(input: &str, output_dir: &str) -> Result<(), CompressError>;

6. Error Handling

pub enum CompressError {
    Io(std::io::Error),
    InvalidInput(Text),
    OutOfBounds,
    InternalError(Text),
    CompressionFailed(Text),
    DecompressionFailed(Text),
    UnsupportedFeature(Text),
}

7. Testing

  • Round‑trip for each algorithm.
  • Empty input.
  • Streaming write/read.
  • Corrupted stream → error.
  • Archive creation/extraction.
