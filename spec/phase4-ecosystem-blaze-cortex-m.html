Blaze Phase 4 – Ecosystem: Cortex‑M

Previous: Config | Next: Crypto

Goal

HAL for ARM Cortex‑M microcontrollers (M0, M3, M4, M7, M33) – NVIC, SysTick, GPIO, USART, SPI, I2C, ADC, PWM. #![no_std], linear types, hal effect.

1. Core Concepts

  • CortexM context holding NVIC and SysTick.
  • SysTick – implements Timer.
  • NVIC – implements InterruptController.
  • GpioPort, GpioPin – GPIO.
  • Usart – serial.
  • Spi, I2c, Adc, Pwm peripherals.

2. CortexM Context

pub struct CortexM { pub systick: SysTick, pub nvic: NVIC }
impl CortexM { pub fn new() -> CortexM; pub fn reset(&self); pub fn disable_interrupts() -> bool; pub fn enable_interrupts(); }

3. SysTick Timer

24‑bit counter; methods: now, delay_us, delay_ms, set_alarm, clear_alarm.

4. NVIC

Methods: enable_irq, disable_irq, set_priority, pending_irq, end_of_interrupt.

5. GPIO

pub struct GpioPort { base: *mut GpioRegisters, port_letter: char }
impl GpioPort { pub fn pin(&self, pin: u8, mode: PinMode) -> GpioPin; }

impl GpioPin { pub fn set_high(&mut self); pub fn set_low(&mut self); pub fn toggle(&mut self); pub fn is_high(&self) -> bool; pub fn is_low(&self) -> bool; pub fn set_mode(&mut self, mode: PinMode); }
