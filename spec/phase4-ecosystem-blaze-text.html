Blaze Phase 4 – Ecosystem: Text

Previous: SVG | Next: TLS

Goal

GPU‑accelerated text layout and rendering engine. Font loading, glyph caching (atlas), shaping (HarfBuzz or pure‑Blaze fallback), measurement, drawing.

1. Font

pub struct Font { family: String, weight: FontWeight, style: FontStyle, data: Vec<u8>, units_per_em: u32, ascent: f64, descent: f64, line_gap: f64, glyph_count: u32 }
pub fn load_system_font(family: &str, weight: FontWeight, style: FontStyle) -> Result<Font, TextError>;
pub fn load_from_file(path: &str) -> Result<Font, TextError>;
pub fn load_from_bytes(bytes: &[u8]) -> Result<Font, TextError>;

2. FontAtlas

pub struct FontAtlas { texture: wgpu::Texture, entries: Map<u32, GlyphEntry>, glyphs: Vec<GlyphCacheInfo>, next_x: u32, next_y: u32, max_row_height: u32 }
pub fn new(font: &Font, size: f64, gpu_device: &GpuDevice) -> FontAtlas;

3. TextLayout

pub struct TextLayout { glyphs: Vec<PositionedGlyph>, total_width: f64, total_height: f64 }
pub struct PositionedGlyph { pub glyph_id: u32, pub x: f64, pub y: f64, pub advance_width: f64 }
pub fn layout_text(font: &Font, atlas: &mut FontAtlas, text: &str, font_size: f64, direction: TextDirection) -> TextLayout;

4. Measurement

pub fn measure_text(font: &Font, text: &str, font_size: f64) -> (f64, f64);

5. Rendering

pub fn draw_text(render_pass: &mut wgpu::RenderPass, layout: &TextLayout, atlas: &FontAtlas, position: (f64, f64), color: Color);
pub fn prepare_text_pipeline(device: &wgpu::Device, config: &wgpu::SurfaceConfiguration) -> wgpu::RenderPipeline;

6. Error Handling

pub enum TextError { Io(std::io::Error), InvalidFont(Text), FontNotFound, GlyphNotFound, AtlasFull }

7. Testing

  • Font loading.

  • Rendering (off‑screen).
