Blaze Phase 4 – Ecosystem: Config

Previous: Compress | Next: Cortex‑M

Goal

Layered configuration system with @config macro, sources (files, environment, CLI), and typed struct merging.

1. @config Macro

@config
struct AppConfig {
    pub host: String,
    pub port: u16,
    pub database_url: String,
    pub max_connections: u32 = 100,
    pub enable_caching: bool = true,
    pub log_level: String = "info",
    pub rate_limit: Option<f64>,
}

Generates ConfigLoader and ConfigError. Field attributes: @config(name), @config(default), @config(env_only), @config(file_only), @config(deserialize_with), @config(merge).

2. Configuration Sources

pub enum ConfigSource {
    File(Path),
    Environment(Text),
    CommandLine,
    Map(Map<Text, Text>),
    Defaults,
}

3. ConfigLoader<T>

pub struct ConfigLoader<T: Config> { sources: Vec<ConfigSource>, override_map: Map<Text, Text> }
impl<T: Config + DeserializeOwned> ConfigLoader<T> {
    pub fn new() -> Self;
    pub fn add_source(mut self, source: ConfigSource) -> Self;
    pub fn set_override(mut self, key: &str, value: &str) -> Self;
    pub fn build(self) -> Result<T, ConfigError>;
}

4. Environment and CLI

Environment variables: prefix (default struct name in SCREAMING_SNAKE_CASE) + field. CLI: kebab‑case names, boolean flags with --no-.

5. File Formats

Uses blaze‑serde; format determined by extension (.toml, .yaml, .json).

6. Error Handling

pub enum ConfigError {
    Io(std::io::Error),
    FileFormat(Text),
    MissingField(Text),
    InvalidValue(Text),
    Deserialization(Text),
    EnvVarNotFound(Text),
    CommandLineParse(Text),
    DuplicateSource(Text),
}
