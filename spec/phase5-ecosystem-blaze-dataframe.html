Blaze Phase 5 – Ecosystem: Dataframe

Previous: Autobalance | Next: ML

Goal

High‑performance, columnar data frame library (inspired by Pandas/Polars) built on blaze‑tensor, blaze‑serde, and ECS storage model.

1. Core Types

pub struct DataFrame { columns: Map<Text, Column>, row_count: usize }
pub struct Series<T: SeriesElement> { data: Tensor<T, 1>, name: Text }
pub enum Column { Int8(Series<i8>), Int16(Series<i16>), … Float32(Series<f32>), Bool(Series<bool>), String(Series<Text>), DateTime(Series<DateTime>), Categorical(Series<Categorical>), List(Box<Column>), Struct(Map<Text, Box<Column>>) }

2. DataFrame Operations

impl DataFrame {
    pub fn new() -> Self;
    pub fn from_columns(columns: Vec<(Text, Column)>) -> Result<Self, DataError>;
    pub fn with_column(&mut self, name: &str, column: Column) -> &mut Self;
    pub fn row_count(&self) -> usize;
    pub fn column_names(&self) -> Vec<&str>;
    pub fn get_column<T: SeriesElement>(&self, name: &str) -> Option<Series<T>>;
    pub fn drop_column(mut self, name: &str) -> Self;
    pub fn slice(&self, range: Range<usize>) -> DataFrameView;
    pub fn head(&self, n: usize) -> DataFrameView;
    pub fn filter(&self, predicate: impl Fn(&DataFrameRow) -> bool) -> DataFrame;
    pub fn select(&self, columns: &[&str]) -> DataFrameView;
    pub fn group_by(&self, by: &[&str]) -> GroupBy;
    pub fn agg(&self, agg_exprs: &[(Text, Aggregation)]) -> DataFrame;
    pub fn sort(&self, by: &[&str], ascending: bool) -> DataFrame;
    pub fn join(&self, other: &DataFrame, on: &[&str], how: JoinType) -> DataFrame;
}

3. GroupBy

pub struct GroupBy { df: DataFrame, groups: Map<GroupKey, Vec<usize>> }
impl GroupBy {
    pub fn agg(self, agg_exprs: &[(Text, Aggregation)]) -> DataFrame;
    pub fn apply(self, f: impl Fn(DataFrameView) -> DataFrame) -> DataFrame;
}
