Blaze Phase 4 – Ecosystem: Embedded

Previous: ECS | Next: ESP32

Goal

Foundational layer for bare‑metal and resource‑constrained embedded systems: #![no_std] runtime, panic handler, region‑based allocator, cooperative actor scheduler, and hardware abstraction traits (GPIO, Serial, Timer, InterruptController).

1. Core Concepts

  • #![no_std] and #![no_runtime].
  • Minimal panic handler (halts).
  • StaticArena bump allocator.
  • Cooperative scheduler for actors.
  • Traits: GpioPin, Serial, Timer, InterruptController.

2. Allocator

pub struct StaticArena { start: *mut u8, end: *mut u8, current: AtomicPtr<u8> }
impl StaticArena { pub const fn new(start: *mut u8, size: usize) -> Self; pub unsafe fn allocate(&self, layout: Layout) -> *mut u8; pub fn reset(&self); }

3. Hardware Abstraction Traits

pub trait GpioPin { fn set_high(&mut self); fn set_low(&mut self); fn toggle(&mut self); fn is_high(&self) -> bool; fn is_low(&self) -> bool; fn set_mode(&mut self, mode: PinMode); }
pub trait Serial { fn write(&mut self, byte: u8) -> Result<(), SerialError>; fn read(&mut self) -> Result<u8, SerialError>; fn write_all(&mut self, data: &[u8]) -> Result<(), SerialError>; fn set_baud_rate(&mut self, baud: u32); }
pub trait Timer { fn now(&self) -> u64; fn delay_us(&mut self, us: u64); fn delay_ms(&mut self, ms: u64); fn set_alarm(&mut self, ticks: u64, callback: fn()); fn clear_alarm(&mut self); }
pub trait InterruptController { fn enable_irq(&self, irq: u16); fn disable_irq(&self, irq: u16); fn set_priority(&self, irq: u16, priority: u8); fn pending_irq(&self) -> Option<u16>; fn end_of_interrupt(&self, irq: u16); }

4. Scheduler

pub struct Scheduler { actors: Vec<SpawnedActor> }
impl Scheduler { pub fn new() -> Self; pub fn spawn<A: Actor>(&mut self, actor: A) -> Result<Capability<A>, SchedulerError>; pub fn tick(&mut self) -> bool; pub fn run(&mut self) -> !; }

5. Panic Handler

Default #[panic_handler] calls HAL's halt(). Can be overridden.

6. Startup

#[entry] macro initialises allocator, calls main (actor).

7. Error Handling

pub enum EmbeddedError {
    OutOfMemory,
    InvalidAlignment,
    SerialError(SerialError),
    NoInterrupt,
    SchedulerFull,
    ActorPanicked,
}

8. Testing

Uses mock HAL via feature mock‑hal. Tests: allocator, GPIO mock, Serial mock, scheduler, panic handler.
