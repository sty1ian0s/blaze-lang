Blaze Phase 4 – Ecosystem: YAML

Previous: WGPU | Next: Zstd

Goal

Fast, zero‑copy YAML parser and writer (v1.2) with serde integration.

1. YAML Value

pub enum YamlValue {
    Null, Bool(bool), Int(i64), Float(f64), Str(Text), Bytes(Vec<u8>),
    Seq(Vec<YamlValue>), Map(Map<Text, YamlValue>),
    Alias(usize), Tagged(Text, Box<YamlValue>), Anchor(usize, Box<YamlValue>),
}

2. Parsing

pub fn from_str(source: &str) -> Result<YamlValue, YamlError>;
pub fn from_bytes(bytes: &[u8]) -> Result<YamlValue, YamlError>;

Handles flow/block style, anchors, aliases, tags, literal/folded scalars, implicit typing.

3. Writing

pub fn to_string(value: &YamlValue) -> Text;
pub fn to_string_pretty(value: &YamlValue) -> Text;
pub fn to_writer<W: Write>(writer: &mut W, value: &YamlValue) -> Result<(), io::Error>;

4. Serde Integration

pub struct Serializer<W: Write> { /* … */ }
impl<W: Write> serde::Serializer for Serializer<W> { … }
pub struct Deserializer<'de> { /* … */ }
impl<'de> serde::Deserializer<'de> for Deserializer<'de> { … }

5. Error Handling

pub struct YamlError { msg: Text, line: usize, column: usize }
impl YamlError { pub fn line(&self) -> usize; pub fn column(&self) -> usize; pub fn message(&self) -> &str; }

6. Testing

  • Valid YAML: various styles.
  • Anchors/aliases.
  • Scalar types.
  • Error cases.
  • Serde integration.
  • Round‑trip.
