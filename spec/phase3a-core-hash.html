Blaze Phase 3a – Core Library: Hashing

Previous: Formatting | Next: I/O

Goal

Provide Hash and Hasher traits, and a default hasher (DefaultHasher).

1. Traits

1.1 Hash

pub trait Hash {
    fn hash<H: Hasher>(&self, state: &mut H);
}

Feeds bytes/structured data into Hasher.

1.2 Hasher

pub trait Hasher {
    fn write(&mut self, bytes: &[u8]);
    fn finish(&self) -> u64;
}

Accumulates bytes; finish returns final hash.

2. DefaultHasher

pub struct DefaultHasher { state: u64 }
impl DefaultHasher {
    pub fn new() -> Self;
}
impl Hasher for DefaultHasher {
    fn write(&mut self, bytes: &[u8]) {
        for &b in bytes {
            self.state = self.state.wrapping_mul(31).wrapping_add(b as u64);
        }
    }
    fn finish(&self) -> u64 { self.state }
}

3. Standard Implementations

  • Integers: feed byte representation.
  • Floats: feed raw bytes (including NaN patterns).
  • bool: feed 0 or 1.
  • char: feed 4‑byte scalar.

@derive(Hash) feeds each field in order.

4. Testing

  • DefaultHasher produces deterministic results for known byte sequences.
  • Equal values produce same hash; different values (usually) produce different hashes.
  • Strings: identical strings hash identically.
  • Incremental hashing works.
