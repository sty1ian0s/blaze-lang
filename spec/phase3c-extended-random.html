Blaze Phase 3c – Extended Library: Random

Previous: Meta | Next: Regex

Goal

Provide traits for random number generators, a thread‑local source of randomness, and functions for generating random values of primitive types. All generators are deterministic given a seed.

1. Rng Trait

pub trait Rng {
    fn next_u32(&mut self) -> u32;
    fn next_u64(&mut self) -> u64;
}

Provided methods: gen_range, gen_range_f, fill_bytes, gen::<T: Random>.

Random Trait

pub trait Random {
    fn random<R: Rng>(rng: &mut R) -> Self;
}

Implemented for all primitive integer/floating types, bool, char, ().

2. SeedableRng Trait

pub trait SeedableRng: Rng {
    fn from_seed(seed: u64) -> Self;
    fn seed(&self) -> u64;
}

3. Default Generator

DefaultRng – cryptographically secure if hardware RNG available, else fast non‑crypto fallback. thread_rng() -> impl Rng returns thread‑local generator.

4. Additional PRNGs

  • Lcg128 – linear congruential generator with 128‑bit state.
  • Xoshiro256StarStar – fast high‑quality xoshiro256** algorithm.

5. Testing

  • Determinism: known seed → known sequence.
  • thread_rng independent per thread.
  • gen_range bounds checked; uniform distribution (chi‑square test).
  • fill_bytes fills correctly.
