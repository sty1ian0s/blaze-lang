Blaze Phase 4 – Ecosystem: Crypto

Goal

Comprehensive cryptographic primitives: secure random, hashing (SHA‑2, SHA‑3, BLAKE3), HMAC, AEAD (AES‑GCM, ChaCha20‑Poly1305), asymmetric (RSA, ECC), key derivation (HKDF, PBKDF2). Linear secret keys, constant‑time operations.

2. Hashing

pub trait Hash {
    fn update(&mut self, data: &[u8]);
    fn finalize(self) -> Vec<u8>;
    fn output_size() -> usize;
    fn block_size() -> usize;
}
pub struct Sha256; pub struct Sha512; pub struct Sha3_256; pub struct Sha3_512; pub struct Blake3;

One‑shot convenience functions: sha256, sha512, etc.

8. Testing

  • Hashing: test vectors.
  • HMAC: RFC 4231.
  • AEAD: encrypt/decrypt round‑trip; tampered ciphertext fails.
  • RSA: keygen, sign/verify.
  • ECC: sign/verify, X25519 DH.
  • HKDF/PBKDF2: test vectors.
  • Zeroisation: private keys zeroed on drop.
