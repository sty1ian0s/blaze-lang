Blaze Phase 3d – Specialized Library: Localisation

Previous: GPU | Next: Accessibility

Goal

Define optional std::l10n module for locale‑specific formatting, message translation, plural rules, and resource bundles.

1. Locale

pub struct Locale {
    language: Text,
    region: Option<Text>,
    script: Option<Text>,
    variant: Option<Text>,
}
impl Locale {
    pub fn new(language: &str) -> Locale;
    pub fn with_region(language: &str, region: &str) -> Locale;
    pub fn from_tag(tag: &str) -> Result<Locale, Error>;
    pub fn default_locale() -> Locale;
    pub fn to_tag(&self) -> Text;
}

2. Message Formatting

pub struct Message { locale: Locale, text: Text }
impl Message {
    pub fn new(locale: &Locale, pattern: &str) -> Result<Message, Error>;
    pub fn format(&self, args: &[impl Display]) -> Text;
}

Supports ICU MessageFormat subset: placeholders {0}, number/plural/select forms.

3. Number and Date Formatting

pub struct NumberFormatter { locale: Locale, style: NumberStyle }
impl NumberFormatter { pub fn format_i64(&self, value: i64) -> Text; pub fn format_f64(&self, value: f64) -> Text; }
pub enum NumberStyle { Decimal, Percent, Currency(Text), Scientific }
pub struct DateFormatter { locale: Locale, date_style: DateStyle, time_style: TimeStyle }
impl DateFormatter { pub fn format(&self, timestamp: i64) -> Text; }
pub enum DateStyle { Full, Long, Medium, Short, None }
pub enum TimeStyle { Full, Long, Medium, Short, None }

4. Plural Rules

pub struct PluralRule { locale: Locale }
impl PluralRule { pub fn select(&self, n: f64) -> PluralCategory; }
pub enum PluralCategory { Zero, One, Two, Few, Many, Other }

5. Resource Bundles

pub struct Bundle { locale: Locale, entries: Map<Text, Text> }
impl Bundle { pub fn load_from_file(path: &str) -> Result<Bundle, Error>; pub fn get(&self, key: &str) -> Option<&str>; }

@bundle("path.json") attribute embeds bundle at compile time.

6. Testing

  • Locale construction/parsing.
  • Message formatting with plural/select.
  • Number formatting for "en‑US" (grouping, decimal).
  • Date formatting (consistent timezone).
  • Plural rule selection (English: one for 1, other for 0,2,…).
  • Bundle loading and retrieval.
