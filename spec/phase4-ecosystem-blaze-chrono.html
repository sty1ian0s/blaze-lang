Blaze Phase 4 – Ecosystem: Chrono

Previous: Bench Utils | Next: CLI

Goal

Comprehensive date and time library with timezone‑aware types, formatting/parsing, calendar arithmetic, and serde integration.

1. Core Types

pub struct NaiveDateTime { pub date: NaiveDate, pub time: NaiveTime }
pub struct NaiveDate { year: i32, ordinal: u16 }
pub struct NaiveTime { secs: u32, nanos: u32 }
pub struct DateTime<Tz: TimeZone> { pub naive: NaiveDateTime, pub timezone: Tz }

2. Timezones

  • Utc, FixedOffset, Local, IanaTimeZone (feature `iana`).

3. Parsing and Formatting

impl NaiveDateTime { pub fn format(&self, fmt: &str) -> Text; }
pub fn parse_iso8601(s: &str) -> Result<NaiveDateTime, ChronoError>;
pub fn to_iso8601(&self) -> Text;

4. Arithmetic

NaiveDateTime + Duration = NaiveDateTime; NaiveDateTime + Months = NaiveDateTime (calendar arithmetic).

5. Serde Integration

All types implement Serialize/Deserialize (ISO strings for human‑readable, binary for compact).

6. Error Handling

pub enum ChronoError {
    Parse(Text),
    OutOfRange,
    InvalidTimezone,
    Io(std::io::Error),
}

7. Testing

  • Construction and weekday.
  • Arithmetic across month boundaries.
  • Parsing ISO strings.
  • Timezone conversion.
