Blaze Phase 3d – Specialized Library: GPU Abstractions

Previous: SIMD | Next: Localisation

Goal

Define optional std::gpu module for GPU programming (device selection, memory allocation, kernel compilation, synchronisation). All operations carry gpu effect.

1. GPU Device Selection

pub struct Device { /* opaque */ }
impl Device {
    pub fn count() -> usize;
    pub fn get(index: usize) -> Option<Device>;
    pub fn name(&self) -> &str;
    pub fn memory_total(&self) -> usize;
    pub fn memory_free(&self) -> usize;
}

2. GPU Memory

pub struct GpuBuffer<T> {
    device: Device,
    ptr: *mut T,
    len: usize,
}
impl<T> GpuBuffer<T> {
    pub fn new(device: &Device, count: usize) -> Result<GpuBuffer<T>, Error>;
    pub unsafe fn from_raw(device: &Device, ptr: *mut T, len: usize) -> GpuBuffer<T>;
    pub fn len(&self) -> usize;
    pub fn as_ptr(&self) -> *const T;
    pub fn as_mut_ptr(&mut self) -> *mut T;
    pub fn copy_from_host(&mut self, host_data: &[T]) -> Result<(), Error>;
    pub fn copy_to_host(&self, host_data: &mut [T]) -> Result<(), Error>;
}
impl<T> Dispose for GpuBuffer<T> { ... }

3. GPU Kernels

pub struct Kernel { /* opaque */ }
impl Kernel {
    pub fn from_ptx(device: &Device, ptx: &str, entry: &str) -> Result<Kernel, Error>;
    pub fn launch(&self, args: &[KernelArg], grid: (u32, u32, u32), block: (u32, u32, u32)) -> Result<(), Error>;
}
pub enum KernelArg<'a> {
    Buffer(&'a dyn GpuBufferAny),
    Int32(i32),
    Float32(f32),
}

4. Synchronization

pub fn sync() -> Result<(), Error>;

Blocks CPU until all GPU operations complete on current device.

5. Error Handling

pub struct Error(GpuErrorKind, Text);
pub enum GpuErrorKind {
    OutOfMemory, InvalidDevice, KernelCompilationFailed,
    LaunchFailed, TransferFailed, Timeout,
}

6. Implementation Notes

  • Uses CUDA driver API (NVIDIA), Vulkan compute, or OpenCL.
  • On unsupported platforms, returns InvalidDevice.
  • Kernel compilation from source not part of this module; runtime loads compiled binaries.

7. Testing

  • Device enumeration (if GPU present).
  • Allocate buffer, copy host→device→host, verify.
  • Kernel launch (minimal PTX) and verify result.
  • Synchronisation: after launch, sync() completes.
