Blaze Phase 4 – Ecosystem: TOML

Previous: TLS | Next: Vulkan

Goal

Fast, zero‑copy TOML parser and writer (v1.0) with serde integration.

1. TOML Value

pub enum TomlValue {
    String(Text),
    Integer(i64),
    Float(f64),
    Boolean(bool),
    Datetime(TomlDatetime),
    Array(Vec<TomlValue>),
    Table(Map<Text, TomlValue>),
}
pub struct TomlDatetime(Text);
impl TomlDatetime {
    pub fn to_string(&self) -> &str;
    pub fn to_system_time(&self) -> Result<SystemTime, TomlError>;
}

2. Parsing

pub fn from_str(source: &str) -> Result<TomlValue, TomlError>;
pub fn from_bytes(bytes: &[u8]) -> Result<TomlValue, TomlError>;

3. Writing

pub fn to_string(value: &TomlValue) -> Text;
pub fn to_string_pretty(value: &TomlValue) -> Text;
pub fn to_writer<W: Write>(writer: &mut W, value: &TomlValue) -> Result<(), io::Error>;

4. Serde Integration

pub struct Serializer<W: Write> { /* … */ }
impl<W: Write> serde::Serializer for Serializer<W> { … }
pub struct Deserializer<'de> { /* … */ }

5. Error Handling

pub struct TomlError { msg: Text, line: usize, column: usize }
impl TomlError { pub fn line(&self) -> usize; pub fn column(&self) -> usize; pub fn message(&self) -> &str; }

6. Testing

  • Valid TOML: all data types.
  • Round‑trip.
  • Invalid TOML: errors with location.
  • Datetime round‑trip.
  • Serde integration.
