Blaze Phase 4 – Ecosystem: Serde

Previous: Redis | Next: Smol

Goal

Data‑oriented serialization/deserialization framework with traits, derive macros, and JSON/binary formats.

1. Core Traits

pub trait Serialize { fn serialize<S: Serializer>(&self, serializer: S) -> Result<S::Ok, S::Error>; }
pub trait Deserialize<'de>: Sized { fn deserialize<D: Deserializer<'de>>(deserializer: D) -> Result<Self, D::Error>; }

2. Serializer and Deserializer

pub trait Serializer: Sized { type Ok; type Error: Error; /* … serialize_bool, serialize_i8, …, serialize_struct, … */ }
pub trait Deserializer<'de>: Sized { type Error: Error; /* … deserialize_bool, deserialize_i8, …, deserialize_struct, … */ }
pub trait Visitor<'de>: Sized { type Value; fn expecting(&self, formatter: &mut Formatter) -> fmt::Result; /* … visit_bool, visit_i8, …, visit_seq, visit_map, … */ }

3. JSON Support

pub mod serde_json {
    pub struct Serializer<W: Write> { /* … */ }
    pub struct Deserializer<'de> { /* … */ }
    pub fn to_string<T: Serialize>(value: &T) -> Result<Text>;
    pub fn from_str<'a, T: Deserialize<'a>>(s: &'a str) -> Result<T>;
}

4. Binary Support

pub mod serde_binary {
    pub struct Serializer<W: Write> { /* … */ }
    pub struct Deserializer<'de> { /* … */ }
}

5. Derive Macros

@derive(Serialize, Deserialize)
struct Point { x: f64, y: f64 }

6. Error Handling

pub struct Error { message: Text, line: Option<usize>, column: Option<usize> }
impl Error { pub fn new(msg: &str) -> Self; pub fn with_location(mut self, line: usize, col: usize) -> Self; }

7. Testing

  • Round‑trip: JSON, binary.
  • Edge cases: large ints, floats, nested structures.
  • Malformed JSON → error.
  • Derive correctness.
