Blaze Phase 3a – Core Library: Default

Previous: Debug | Next: Environment

Goal

Implement Default trait for creating default values.

1. Default Trait

pub trait Default {
    fn default() -> Self;
}
  • Used by @derive(Default) and generic code.
  • For structs with field defaults, compiler provides Default automatically if all fields have defaults; otherwise must be manual or derived.

2. Standard Implementations

  • Integers → 0; floats → 0.0; bool → false; char → '\0'; Option<T> → None.
  • Text → Text::new(); Vec<T> → empty.
  • Result does not implement Default.

3. Testing

  • Each primitive type's default matches expected zero/false.
  • Option::<i32>::default() is None.
  • Text::default() is empty string.
  • Derived Default on struct with field defaults.
