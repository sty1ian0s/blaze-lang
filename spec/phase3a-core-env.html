Blaze Phase 3a – Core Library: Environment

Previous: Default | Next: Formatting

Goal

Provide access to environment variables, current working directory, and program arguments.

1. Functions

1.1 var

pub fn var(key: &str) -> Result<Text, VarError>;

Retrieves environment variable; returns VarError if not found or invalid UTF‑8.

1.2 set_var

pub fn set_var(key: &str, value: &str);

Sets environment variable; panics on failure.

1.3 remove_var

pub fn remove_var(key: &str);

Removes environment variable; panics on failure.

1.4 current_dir

pub fn current_dir() -> Result<Text, io::Error>;

Returns current working directory.

1.5 args

pub fn args() -> Vec<Text>;

Returns command‑line arguments (including program name).

2. VarError

pub struct VarError { /* private */ }

Provides to_text() for human‑readable description.

3. Testing

  • var: set and retrieve; missing var returns error.
  • set_var and remove_var: basic cycle.
  • current_dir: returns existing directory.
  • args: test with known arguments.
