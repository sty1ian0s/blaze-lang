Blaze Phase 5 – Ecosystem: Asset

Previous: Zstd | Next: Autobalance

Goal

Compile‑time and runtime asset pipeline for textures, meshes, audio, and arbitrary data. Integrates with @texture, @generate, @streaming, and @derive(Deduplicate).

1. Core Types

pub struct Texture { /* GPU handle */ }
pub struct Mesh { /* vertex/index buffers */ }
pub struct AudioClip { /* compressed audio */ }
pub struct AssetCache { /* streaming cache */ }
pub struct AssetId { /* typed identifier */ }

2. @texture Attribute

@texture(compression = "auto", format = "bc7", mipmaps = "generate", srgb = true)
static DIFFUSE_MAP: Texture = include_texture!("wall_diffuse.png");

Compresses texture at build time to GPU‑native format (BCn, ASTC). Stores directly in binary; no runtime decompression.

3. @generate Attribute

@generate(from = "diffuse.png", algorithm = "sobel") static NORMAL_MAP: Texture;
@generate(algorithm = "terrain", seed = 42, size = 1024) static TERRAIN_HEIGHTMAP: Texture;

Runs procedural generation at compile time. Built‑in algorithms: sobel, normal_from_height, mipmap, lod, terrain, cube_from_2d.

4. @streaming Attribute

@streaming(threshold = "4MB", prefetch = 2) static LEVEL_DATA: &[u8] = include_bytes!("level.bin");

Marks asset for automatic runtime paging. StreamingSlice<T> with get (non‑blocking) and fetch (async).

5. @derive(Deduplicate)

Globally deduplicates constant instances of the type at compile time. Used by asset types to eliminate duplicate textures, meshes, materials.

6. Runtime Asset Cache

pub struct AssetCache { /* … */ }
impl AssetCache {
    pub fn new(config: CacheConfig) -> Self;
    pub fn get<T: Asset>(&self, id: &AssetId) -> Option<Arc<T>>;
    pub fn insert<T: Asset>(&self, id: AssetId, asset: T) -> Arc<T>;
    pub fn evict(&self, id: &AssetId);
}

7. Error Handling

pub enum AssetError { Io(std::io::Error), InvalidFormat(Text), GenerationFailed(Text), OutOfMemory, StreamingNotAvailable }

8. Testing

  • Texture compilation: BC7 format and mipmaps.
  • Deduplication: two identical constants → one copy.
  • Generation: sobel produces normal map.
  • Streaming: pages loaded on demand; prefetch works.
  • Asset cache: insert/retrieve/evict.
