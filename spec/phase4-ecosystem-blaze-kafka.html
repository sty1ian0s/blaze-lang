Blaze Phase 4 – Ecosystem: Kafka

Previous: JWT | Next: Layout

Goal

Async Kafka client with native wire protocol support for producers, consumers, consumer groups, and administrative operations. Built on Blaze's async I/O and actor model.

1. Core Types

pub struct KafkaClient { /* … */ }
pub struct Producer<K, V> { /* … */ }
pub struct Consumer<K, V> { /* … */ }
pub struct KafkaRecord<K, V> { pub topic: Text, pub partition: i32, pub offset: Offset, pub timestamp: i64, pub key: Option<K>, pub value: V, pub headers: Vec<(Text, Vec<u8>)> }
pub struct TopicPartition { pub topic: Text, pub partition: i32 }
pub struct Offset(i64);

2. Client

impl KafkaClient {
    pub async fn connect(brokers: &str, config: &KafkaConfig) -> Result<Self, KafkaError>;
    pub fn create_producer<K: Serialize, V: Serialize>(&self, topic: &str) -> Producer<K, V>;
    pub fn create_consumer<K: Deserialize<'static>, V: Deserialize<'static>>(&self, topic: &str, group: &str) -> Consumer<K, V>;
    pub fn admin(&self) -> AdminClient;
}

3. Producer

impl<K: Serialize, V: Serialize> Producer<K, V> {
    pub async fn send(&self, record: KafkaRecord<K, V>) -> Result<(), KafkaError>;
    pub fn send_batch(&self, records: Vec<KafkaRecord<K, V>>) -> Result<(), KafkaError>;
    pub async fn flush(&self);
    pub async fn close(self);
}

4. Consumer

impl<K: Deserialize<'static>, V: Deserialize<'static>> Consumer<K, V> {
    pub async fn recv(&self) -> Option<KafkaRecord<K, V>>;
    pub async fn commit(&self);
    pub async fn seek(&self, tp: TopicPartition, offset: Offset);
    pub async fn close(self);
}

5. Admin

impl AdminClient {
    pub async fn create_topic(&self, name: &str, partitions: i32, replication_factor: i16) -> Result<(), KafkaError>;
    pub async fn delete_topic(&self, name: &str) -> Result<(), KafkaError>;
    pub async fn list_topics(&self) -> Result<Vec<TopicInfo>, KafkaError>;
}

6. Wire Protocol

Implements API keys: Metadata, Produce, Fetch, OffsetFetch, OffsetCommit, GroupCoordinator, JoinGroup, SyncGroup, Heartbeat, LeaveGroup, ApiVersions, CreateTopics, DeleteTopics.

7. Error Handling

pub enum KafkaError {
    Io(std::io::Error),
    Protocol(Text),
    ApiVersionNotSupported,
    CoordinatorNotAvailable,
    NotCoordinator,
    OffsetOutOfRange,
    UnknownTopicOrPartition,
    LeaderNotAvailable,
    NotLeaderForPartition,
    RequestTimedOut,
    ConsumerRebalanceInProgress,
    Serialization(Text),
    Deserialization(Text),
    AuthenticationFailed,
    AuthorizationFailed,
}
