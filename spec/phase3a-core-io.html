Blaze Phase 3a – Core Library: I/O

Previous: Hashing | Next: Iterators

Goal

Implement Read and Write traits, File, Metadata, standard streams, and error types.

1. Traits

1.1 Read

pub trait Read {
    fn read(&mut self, buf: &mut [u8]) -> Result<usize, Error>;
    fn read_to_end(&mut self, buf: &mut Vec<u8>) -> Result<usize, Error>;
    fn read_to_text(&mut self) -> Result<Text, Error>;
}

1.2 Write

pub trait Write {
    fn write(&mut self, buf: &[u8]) -> Result<usize, Error>;
    fn flush(&mut self) -> Result<(), Error>;
}

2. File

pub struct File { fd: i32, is_stdout: bool }
impl File {
    pub fn open(path: &str) -> Result<File, Error>;
    pub fn create(path: &str) -> Result<File, Error>;
    pub fn metadata(&self) -> Result<Metadata, Error>;
    pub fn set_nonblocking(&mut self, nonblocking: bool) -> Result<(), Error>;
}

impl Write for File { ... }
impl Dispose for File { ... }

3. Metadata

pub struct Metadata { size: u64, kind: u8 }
impl Metadata {
    pub fn len(&self) -> u64;
    pub fn is_file(&self) -> bool;
    pub fn is_dir(&self) -> bool;
}

4. Standard Streams

pub struct Stdout { fd: i32 }
pub struct Stderr { fd: i32 }
pub struct Stdin { fd: i32 }
pub fn stdout() -> Stdout;
pub fn stderr() -> Stderr;
pub fn stdin() -> Stdin;

Implement Write for Stdout and Stderr; Read for Stdin.

5. Error Handling

pub struct Error(ErrorKind, Text);
pub enum ErrorKind {
    NotFound, PermissionDenied, ConnectionRefused, ConnectionReset,
    Interrupted, UnexpectedEof, InvalidInput, TimedOut, Other
}

  • Mock Read/Write.
